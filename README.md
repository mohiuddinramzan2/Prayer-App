# 🕌 নামাজের সময় — Prayer App v2

## ✨ ফিচার
- 📍 GPS ভিত্তিক সঠিক নামাজের সময়
- 🔊 **আজান** — ফজরে বিশেষ আজান, বাকি ওয়াক্তে সাধারণ আজান
- 🕌 **জুম'আর সময়সূচী** — খুতবা ও নামাজের সময়
- 🌙 **ইফতার ও সাহরীর সময়** — কাউন্টডাউন সহ
- 🧭 **কিবলা কম্পাস** — ডিভাইস সেন্সর ব্যবহার করে
- 📿 **ডিজিটাল তসবিহ** — সুবহানআল্লাহ, আলহামদুলিল্লাহ ও আরো
- 📱 PWA — ফোনে ইনস্টল করা যাবে

## 📁 ফাইল স্ট্রাকচার
```
prayer-app/
├── index.html
├── manifest.json
├── css/style.css
├── js/
│   ├── prayer.js     ← নামাজের সময় গণনা
│   ├── qibla.js      ← কিবলা ও কম্পাস
│   └── app.js        ← UI + আজান + তসবিহ
└── assets/
    ├── azan_fajr.mp3    ← ফজরের আজান (আপনার ফাইল)
    ├── azan_normal.mp3  ← সাধারণ আজান (আপনার ফাইল)
    ├── icon-192.png
    └── icon-512.png
```

## 🚀 GitHub Pages Deploy
1. GitHub-এ repo বানান
2. সব ফাইল push করুন
3. Settings → Pages → Branch: main → Save
4. লাইভ লিংক পাবেন!

## 📱 PWA ইনস্টল
Chrome → মেনু → Add to Home Screen
